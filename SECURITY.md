# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

如果您发现安全漏洞，请通过以下方式报告：

1. **不要**在公开的 Issue 中报告安全问题
2. 发送邮件到：[zyhz.zhang@outlook.com]
3. 包含详细的漏洞描述和重现步骤
4. 我们会在 48 小时内回复

## 安全最佳实践

- 应用只在本地存储数据
- 不进行网络通信
- 遵循 macOS 安全沙盒规则
- 定期更新依赖项
